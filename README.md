# pyfabricconnect
PyFabricConnect  Unified Python connectivity toolkit for Microsoft Fabric Warehouses, Lakehouses, and SQL Analytics Endpoints.  PyFabricConnect provides a simple and extensible way to connect Python applications, Spark workloads, and data engineering pipelines to Microsoft Fabric using modern authentication methods such as Service Principal and OAuth Access Tokens.  The library abstracts JDBC, ODBC, and SQLAlchemy integrations into a unified developer experience, allowing users to seamlessly work with:  Spark DataFrames Pandas DataFrames SQLAlchemy Engines Fabric Warehouses Lakehouse SQL Endpoints  PyFabricConnect is designed for:  Data Engineers Data Analysts Analytics Engineers Python Developers Microsoft Fabric teams  Key features include:  Unified authentication layer JDBC connectivity for Spark ODBC connectivity with pyodbc SQLAlchemy integration OAuth Token support Service Principal authentication Enterprise-ready architecture Extensible connector framework  PyFabricConnect aims to simplify secure connectivity and accelerate development workflows across the Microsoft Fabric ecosystem.
